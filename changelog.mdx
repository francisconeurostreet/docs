<Update label="June 15, 2026" tags={["API"]}>

- Added support for authorizing or revoking market data entitlements for multiple exchanges in a single user entitlement update request using `exchange_codes`.
- Existing single-entitlement requests using `entitlement` continue to work unchanged.

</Update>

<Update label="June 12, 2026" tags={["Admin Dashboard"]}>

- Added CSV export for the admin Users grid, including all users that match the current search, sort, and filters.

</Update>

<Update label="June 11, 2026" tags={["API"]}>

- Added support for enabling L2 market data access directly for a user.

</Update>

<Update label="June 10, 2026" tags={["Admin Dashboard"]}>

- Challenge configuration now shows available market data entitlements while admins edit challenge trading rules.
- Improved admin user list filtering and sorting, including support for user type classification.
- Added an account trade-data download so admins can export closed positions and order history for a selected account.

</Update>

<Update label="June 8, 2026" tags={["Docs"]}>
  Launched the public Ignite API documentation with an authentication quickstart, API reference, and changelog.
</Update>
