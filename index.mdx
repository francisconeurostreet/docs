Ignite API helps partner teams integrate with the operational workflows behind Ignite.

Use these docs to authenticate, manage users and trading accounts, review entitlements, inspect account activity and track public API changes.

## Some of the things you can do

- Authenticate with short-lived session tokens.
- Create, retrieve, deactivate, and reactivate trading accounts.
- Inspect account balances, status, summary data, and position activity.
- Manage users, memberships, notes, and account statistics.
- Review and update market data entitlements.
